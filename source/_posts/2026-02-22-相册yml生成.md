# 引言<br>
起因是我想丢一大堆图片进Redefine主题的瀑布流相册功能里，但是你想添加就只能一张张照片吧名字啥的添加进去`source\_data\masonry.yml`，**灰常麻烦！** 于是作为懒人的我，就直接给AI提要求。


# 准备工作
1. 为了更快地把图片加入`source\_data\masonry.yml` ，建议把照片都放进同一个文件夹先。
2. 新建一个`.bat`文件，然后写入以下代码。
```
@echo off
chcp 65001 >nul
echo Generating masonry.yml...

echo # Masonry gallery configuration > masonry.yml
for %%i in (*.jpg *.png *.jpeg) do (
echo - image: /images/%%i >> masonry.yml
echo   title: >> masonry.yml
echo   description: >> masonry.yml
)

echo Done! Check masonry.yml in current directory.
pause
```


# 一步到位！
最后，在和那一堆照片同目录下，找到你的`masonry.yml`就好了。
> [!note] 温馨提示
> 要记得把照片和.yml文件放到对应位置才部署！
